Visualization of Residuals

# Add fitted values and residuals to the data
atib <- atib %>%
  mutate(fitted = model$fitted.values,
         resid = model$residuals,
         label = paste0("epsilon[", row_number(), "]"))

ggplot(atib, aes(x = x, y = y)) +
  geom_point(color = "red", size = 2) +                         # Original data points
  geom_smooth(method = "lm", se = FALSE, color = "blue") +     # Regression line
  geom_segment(aes(xend = x, yend = fitted),                   # Residual lines

  geom_text(aes(label = label, y = y + 0.5), parse = TRUE) +   # Epsilon labels
  theme_bw() +                                                 # Match theme

What is regression

Regression analysis is a tool to investigate how two or more variables are related.

“The general objective of a regression analysis is to determine the relationship between two (or more) variables so that we can gain information about one of them through knowing values of the other(s).”

Source: Ch 12 of Devore, Berk, and Carlton (2021)

Variable Types

We want to see how a specific response variable, \(Y\) is affected by one or more explanatory variables \(X_1, \dots, X_p\)

  • \(Y\) is also known as the dependent variable or output variable. This variable is continuous (i.e. numeric data type)

  • \(X_i\)’s are also known as independent/predictor variables or features. These may be continuous or categorical; however, we only consider continuous predictors in this course.

Example

One may wish to use a person’s height, gender and race to predict a person’s weight. What is/are the response variable(s)?

Model

\[\begin{align} Y &= \underbrace{f(X_1, \dots, X_p)}_{\text{deterministic}} + \underbrace{\epsilon}_{\text{random}}\\ &= \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots \beta_p X_p + \epsilon \end{align}\]

. . .

In a linear regression model, the function \(f(X_1, \dots, X_p)\) is a linear combination of the explanatory variables.

Model

\[\begin{align} \overset{\overset{\text{response}}{\downarrow}}{Y} &= \beta_0 + \beta_1 \underset{\underset{\text{explanatory}}{\uparrow}}{X_1} + \beta_2 \underset{\underset{\text{explanatory}}{\uparrow}}{X_2} + \dots \beta_p \underset{\underset{\text{explanatory}}{\uparrow}}{X_p} + \epsilon \end{align}\]

  • \(\beta_0\) is the \(y\)-intercept

  • \(\beta_i\) are the regression coefficients, and

  • \(\epsilon\) are the errors that are assumed \(N(0, \sigma)\)

In statistics, we can describe how variables are related using a mathematical function.

The function along with other assumptions is called a model.

There are many models we can consider. In this class, we will focus on linear models, particularly, when there is only one predictor variable.

We refer to this model as the simple linear regression model. (Penn STat 500)

as well as modeling numerical response variables using a numerical or categorical explanatory variable.

Simple Linear Regression

  • When \(p = 1\), namely, we are only considering one explanatory variable, our model is called “simple” linear regression (SLR).
  • When \(p>1\) we fall into multiple linear regression (MLR).

SLR model

The SLR model reduces to an equation for of a line:

\[Y = \beta_0 + \beta_1 X\]

where:

  • \(\beta_0\) is the \(y\)-intercept, and

  • \(\beta_1\) is the slope
